# context/

Esta pasta existe para que o contexto do projeto sobreviva entre conversas novas com o Claude Code (cada chat novo começa "zerado" a menos que estes arquivos sejam lidos).

- **rules.md** — o acordo de trabalho: o que o Claude pode e não pode fazer neste repositório.
- **progress.md** — estado atual do desafio: o que já foi decidido/implementado, o que falta.
- **decisions.md** — decisões técnicas relevantes tomadas durante as consultorias, com o raciocínio por trás.

No início de uma sessão nova, vale pedir para o Claude ler esta pasta antes de continuar a conversa.
